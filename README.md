# Proyecto: Fundamentos de JavaScript

Este proyecto contiene una serie de archivos y ejemplos prácticos para aprender y reforzar los conceptos fundamentales de JavaScript. Está diseñado para estudiantes y desarrolladores que desean comprender la sintaxis, las estructuras de datos, el manejo de objetos, funciones, métodos y el uso del keyword `this` en JavaScript.

## Estructura del Proyecto

- **index.html**: Archivo principal para cargar y probar los scripts JavaScript en el navegador.
- **styles.css**: Hoja de estilos para la presentación visual (opcional).
- **js/**: Carpeta que contiene los ejemplos de código JavaScript organizados por temas:
  - `01Variables.js`: Declaración y uso de variables (`var`, `let`, `const`).
  - `02Primitives.js`: Tipos de datos primitivos (string, number, boolean, null, undefined).
  - `03Array.js`: Creación y manipulación de arreglos.
  - `04Objects.js`: Definición y uso de objetos.
  - `05Conditionals.js`: Estructuras condicionales (`if`, `else`, `switch`).
  - `06ForLoop.js`: Bucles y ciclos (`for`, `while`, `do...while`).
  - `07Functions.js`: Declaración y uso de funciones (tradicionales y flecha).
  - `08MethodsAndThisKeyword.js`: Métodos en objetos y el uso de la palabra clave `this`.

## Ejecución de los Ejemplos

1. Abre el archivo `index.html` en tu navegador preferido.
2. Modifica la referencia al archivo JavaScript que deseas probar en la etiqueta `<script src="js/XXArchivo.js"></script>`.
3. Observa los resultados en la consola del navegador (F12 o Ctrl+Shift+I).

## Descripción de los Temas

### Variables
Aprende a declarar variables usando `let` y `const`, y conoce las diferencias entre ellas.

### Tipos Primitivos
Identifica y utiliza los tipos de datos básicos en JavaScript.

### Arreglos
Crea, accede y modifica arreglos.

### Objetos
Define objetos, accede a sus propiedades y modifica sus valores.

### Condicionales
Implementa lógica condicional para controlar el flujo de tu programa.

### Bucles
Utiliza diferentes tipos de bucles para iterar sobre datos y ejecutar código repetidamente.

### Funciones
Declara funciones tradicionales y funciones flecha, y comprende el alcance de las variables.

### Métodos y la palabra clave `this`
Implementa métodos dentro de objetos y entiende cómo funciona el contexto de `this`.

## Requisitos

- Navegador web moderno (Chrome, Firefox, Edge, etc.)
- Editor de código (Visual Studio Code recomendado)

Este proyecto puede ser utilizado como material de estudio, referencia rápida o base para ejercicios prácticos en cursos de JavaScript.
